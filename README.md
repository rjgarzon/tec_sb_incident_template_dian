#Proyecto integrador
 
Realizar un proyectos enfocado en mostrar DEvOps
 
## Estructura del repositorio

- `/src` - código fuente

- `/docs` - documentación

- `/tests` - pruebas automatizadas
 
## Requisitos

- Java 11

- Maven 3.6+

- Docker (Podman)
 
## Cómo ejecutar (local)

1. `mvn clean install`

2. `mvn spring-boot:run`
 
## Flujo de trabajo Git (resumen)

- Ramas: main / develop / feature/* / release/* / hotfix/*

- Branch protections en `main` y `develop`

- Pull requests con al menos 1 revisor y CI passing
